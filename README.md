# Inventário Híbrido

Landing page inicial do projeto **Inventário Híbrido**, um sistema de inventário offline-first projetado para varejo e atacado.

Este é um projeto arquitetado e pronto para ganhar vida — agora busco desenvolvedores parceiros para transformar essa solução em um produto real.

## Objetivo
Criar um sistema de inventário híbrido (online + offline) rápido, confiável e acessível para pequenos e médios negócios.

## Status do Projeto
🚧 Arquitetado  
🚧 Em fase de busca por parceiros e desenvolvedores  
🚧 Landing page inicial publicada

## Tecnologias (planejadas)
- HTML / CSS / JS
- SQLite / MySQL
- Arquitetura offline-first
- Sincronização inteligente
